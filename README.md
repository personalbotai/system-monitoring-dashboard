# System Monitoring Dashboard

Monitor system resources (CPU, Memory, Disk) for distributed systems.

## Features

- Real-time CPU, memory, disk metrics
- Multi-host monitoring
- Historical graphs
- Alert notifications

## Development

This is a static single-page application. Simply open `index.html` in your browser or deploy to GitHub Pages.

## Deployment

GitHub Pages is configured via `.github/workflows/deploy.yml`. After enabling Pages in repository settings, the site will be available at:

```
https://personalbotai.github.io/system-monitoring-dashboard/
```
